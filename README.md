# 🛡️ ADDU Lost & Found
**A campus-centric solution for lost and found management at Ateneo de Davao University.**

ADDU Lost & Found is a mobile application designed to bridge the gap between students who have lost personal belongings and those who have found them. By providing a centralized, real-time platform, the app streamlines the recovery process, reduces campus clutter, and fosters a community of honesty and assistance.

---

## ✨ Features
* **Multi-Method Authentication:** * Secure login using official **ADDU Google Accounts**.
    * Email & Password registration with **6-digit PIN verification** for added security.
    * **Guest Access** for quick browsing of the lost items feed.
* **Real-time Item Feed:** View a live list of reported items directly from the campus database.
* **Visual Verification:** Integrated image support to help users identify their missing items at a glance.
* **Persistent Sessions:** The app remembers your login state so you can stay updated on-the-go.

---

## 🛠️ Tech Stack
* **Frontend:** Flutter & Dart
* **Backend:** Supabase (PostgreSQL Database, Authentication, and Cloud Storage)
* **Real-time:** Supabase Broadcast & Streams for live updates.

## 🚀 Getting Started
To run this project locally for development or testing:

1. **Clone the repository:**
```bash
git clone https://github.com/BrnToLrn/university-lost-and-found.git
```

2. Navigate to the directory
```bash
cd university-lost-and-found
```
   
3. Install dependencies
```bash
flutter pub get
```
   
4. Run the application
```bash
flutter run
```
